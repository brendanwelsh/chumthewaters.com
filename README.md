# chumthewaters.com

Retro game-shark themed linktree page.

**Live:** [chumthewaters.com](https://chumthewaters.com)

## Stack

- Single-page static HTML/CSS/JS
- Custom font (amity-jack.ttf)
- Hosted on [Cloudflare Pages](https://pages.cloudflare.com)
- No build step

## Deploy

```bash
wrangler pages deploy . --project-name=chumthewaters-com
```
